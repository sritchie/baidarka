** Bill of Materials

   Some functions to build up basic units support.

   #+begin_src clojure
;; Units in cm.
(defn cm [x]
  {:value x
   :units :cm})

(defn in [x]
  {:value x
   :units :inch})

(defn tubing [& {:keys [od length] :as m}]
  {:od od
   :length length})
   #+end_src
